## Technology 

Here are the technologies used in this project.

* React native 0.70.5
* TypeScript 4.9.4
* Expo 47.0.8
* Axios 1.2.1
* Styled-components 5.3.6
* asyncStorage 1.17.11


## Getting started

* Install the package.json.
 
* Install Android studio emulator.
  
* To run the project with the command yarn expo start.
